# Make sure you have weechat-python installed

/server add freenode chat.freenode.net
/set irc.server.freenode.nicks "uname"

/set irc.server.freenode.autoconnect on

/set irc.server.freenode.addresses "chat.freenode.net/7000"
/set irc.server.freenode.ssl on
/set irc.server.freenode.sasl_username "mynick"
/set irc.server.freenode.sasl_password "xxxxxxx"

/connect freenode

# PLUGINS:

* VIM in IRC,  https://github.com/GermainZ/weechat-vimode 
* multiline
* autojoin
* Notifications: https://github.com/s3rvac/weechat-notify-send (install dunst and libnotify)
* smart filter: 
	* /set irc.look.smart_filter on 
	* /filter add irc_smart * irc_smart_filter *
